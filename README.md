#Simple Java Web Server
 
### Start Server
 cd my-app

 `java -jar Server.jar -p 5000 -d public`

### Requried Options
 `-p {PORT}`
 `-d {DIRECTORY PATH}`

### Example
 `java -jar Server.jar -p 5000 -d public`

### Resources
 Default directory resources are available in my-app/public

### To Run Test
 cd my-app
 
 mvn clean test
